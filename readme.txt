= Is Transcy Translate free? =
Yes! Transcy Translate is free. You can add one of 110 available languages to your website. If you require the addition of two languages, you can try our Premium plan.


= How do I install Transcy? How do I set up Transcy Translate? =
Transcy Translate is easy to set up:
1. Download the zip folder named transcy.zip
2. Unzip the folder and upload the transcy folder to the ‘/wp-content/plugins/’ directory
3. Activate the plugin through the ‘Plugins’ menu in your WordPress admin
4. Go to the Transcy menu and then choose your settings.
